Sentiment analysis is an interesting thing!

In efforts to put this concept into practice, I devised my own sentiment analysis algorithm in Python using attributes such as words, word lengths, word stems, and sentence lengths to calculate a similarity score all without any use of libraries.

I chose to apply this algorithm to trending hashtags on Twitter and discovered that the algorithm was accurate in determing whether or not certain tweets were associated with positivity or negativity. The text classification tool I built can generally be applied to any model so long as the input is the form of text file.

The most challenging aspect of this project was working with an API in order to access data. I found that in terms of accessing tweets via the Twitter API, the easiest way to do so was to import the tweepy library within Python. This especially made it much easier to access tweets by hastag.
